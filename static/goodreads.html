Crappy Goodreads Unoffical API

Gives JSON data of books' title, authors, cover, pages, ratings

List items:

action: search
Parametres:
q - Search term
filter - Filer
filter can be 0 (all), 1 (title), 2 (author) or 3 (genre). Count - Number of items

example: /search?q=test&filter=0&count=20

Get single book details:

action: view
Parametres:
link - Goodreads link to the book. Can be retrived from the above action
example: /view?link=https://www.goodreads.com/book/show/231804.The_Outsiders
